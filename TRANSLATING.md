# Translate Fluent New Tab

Help us make Fluent New Tab accessible globally! We use **Crowdin** for localization, so you don't need any coding or Git experience to contribute.

[👉 Join our Crowdin Project](https://crowdin.com/project/fluent-new-tab)

---

## Guidelines

To keep translations consistent and functional:

1. **Keep Placeholders Intact:** Never translate or remove variables like `$NAME$`.
   - _Example:_ `Good morning, $NAME$` (Do not change to `Good morning, John`).
2. **Context Matters:** Avoid literal translations. Use clear, universal language suitable for a UI. Keep design terms like **"3D Fluent"** or **"Outline"** in English.
3. **Punctuation & Tone:** Maintain original punctuation (e.g., `...` or `!`). Keep the tone friendly, minimal, and professional.

---

## Missing Your Language?

If your language isn't on Crowdin:

1.  Go to the [Crowdin Project Page](https://crowdin.com/project/fluent-new-tab).
2.  Click **"Request New Language"** or leave a comment on the discussion board.
3.  Or, open a GitHub Issue.

---

## ⚠️ Attention Developers

**Do NOT open Pull Requests to edit translation files (`messages.json`).**

Crowdin is our source of truth. Any manual PR changes to locales will be overwritten by automatic syncs. Please use Crowdin for all text corrections and new translations.
