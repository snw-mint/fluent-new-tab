Theme & Appearance

Located at the very top of the settings menu, you can quickly toggle between three main color schemes for your new tab interface.

Accent Color

You can now replace the default blue accent color across the extension. Choose to automatically extract the accent color directly from your active wallpaper for a dynamic, cohesive look, or pick a specific custom color using the new color matrix.

Light Mode

A clean, bright interface using the standard Fluent Design light palette. Ideal for daytime usage and well-lit environments.

Dark Mode

A deep, high-contrast dark palette that reduces eye strain in low-light environments and makes colors pop.

Auto Mode

Automatically syncs the extension's theme with your operating system's current color scheme (Windows/macOS settings).

Wallpaper

Customize the background of your new tab. If toggled off, the background will revert to a solid color based on your active theme.

Built-in Presets

Choose from three default, high-quality abstract wallpapers carefully designed to match the Fluent Design aesthetic without distracting from your content.

Upload Custom Image

Click the upload icon to set any personal image as your background. Images are automatically optimized, converted to WebP format, and stored securely in your browser's local database.

Dynamic Image Sources

Instead of a static image, you can fetch a new wallpaper automatically every day from external sources: Bing Daily Image, NASA Astronomy Picture of the Day (APOD), or Wikimedia Commons. Credits are shown in the bottom left corner.

Overlay Opacity

Fine-tune how much the wallpaper blends into the background with a dedicated opacity slider ranging from 0% to 60%. Useful for improving text readability over bright or high-contrast images.

Display Widget

The former "Greetings" feature has been completely reformulated into a versatile "Display" module, giving you extensive control over your new tab.

Widget Layouts

Select exactly what you want to see: greetings, a clock (12h or 24h, with or without seconds), a date (numeric or written format), or a combined clock & date view.

Icon/Emoji Visibility

Added a highly requested option to completely remove the icon/emoji from greetings for a cleaner, minimalist aesthetic. Alternatively, you can still use the 3D Fluent emojis or Outline style.

Custom Greetings

Introduced new standard greetings, including custom messages tailored specifically for each day of the week, alongside the dynamic time-of-day greetings.

Weather

A subtle widget displayed in the top-right corner that shows the current temperature and weather conditions for your location, powered by Open-Meteo.

Location Setup

Type the name of your city in the input field and click the save icon. The system will geocode the location and fetch the current meteorological data.

Temperature Unit

Toggle seamlessly between Celsius (°C) and Fahrenheit (°F). The conversion is applied instantly without needing to fetch data again.

Shortcuts

The grid of quick links located right below the search bar. You can add, edit, and drag-and-drop to reorder them directly on the main page.

Grid Rows

Control the maximum amount of shortcuts displayed on the screen. You can limit the grid to show anywhere from 1 to 4 rows. If disabled entirely, the grid is hidden to keep the interface completely minimalist.

Folders

Enable folder support to organize your shortcuts grid into groups. When active, the Add button lets you choose between adding a shortcut or a new folder. Each folder holds up to 39 shortcuts and supports a custom name. Drag shortcuts onto a folder card to move them inside, or drag them back out to the main grid to remove them from the folder.

Drag & Drop from Favorites Bar

Drag bookmarks directly from the browser's favorites bar and drop them into the shortcuts grid or into an open folder. You can also drag a shortcut from the main grid onto a folder card and release to move it inside.

App Launcher

A floating button on the left side of the screen that opens a quick-access drawer to web applications.

Ecosystem Provider

Select which suite of tools you use the most. The launcher will automatically populate with shortcuts and icons for Microsoft 365 (Word, Teams, Copilot), Google Workspace (Docs, Drive, Gemini), or Proton AG (Mail, Calendar, Pass).

Convert App to Folder

Turn any App Launcher app into a shortcut folder for even faster access directly from the shortcuts grid. Click the + icon on an app card to trigger the conversion. The Folders feature must be enabled in the Shortcuts settings for this option to be available.

Search

Customize the behavior and appearance of the central search bar.

Search Engine Provider

Change the default search engine directly on the main page by clicking the engine icon inside the bar to seamlessly switch between Bing, Google, Brave Search, DuckDuckGo, Ecosia Search, Startpage, and Kagi.

Suggestions

When enabled, typing in the search bar will fetch live autocomplete suggestions to help you find what you are looking for faster.

Clear Search (Google)

This setting appends the &udm=14 parameter to Google searches. It forces Google to return a clean, classic list of links, stripping away AI overviews, sponsored widgets, and shopping cards.

Compact Bar

Reduces the height and visual footprint of the search bar for a tighter, more refined aesthetic.

Voice Search

Enables the microphone icon inside the search bar, allowing you to search the web using voice recognition (requires browser permission).

Performance

Fine-tune how the extension renders and animates to best match your hardware and personal preference.

Reduce Motion

Disables all CSS transitions, interface animations, and hover effects. Highly recommended for older computers or for users who prefer an instant, snappy response.

Reduce Blur

Turns off backdrop blur effects used throughout the interface. Blur filters can be GPU-intensive — disabling them noticeably improves rendering performance on lower-end devices.

Utilities

Language settings and data management tools to keep your setup organized and portable.

Language

Set the interface language for the extension menu and greetings. It supports over 20 languages. The default language is English, which is also used as a fallback whenever a translation is missing or unavailable. Found a translation error or want to request a new language? Contribute on Crowdin.

Data Backup (Export / Import)

Allows you to export all your configured settings, shortcuts, and preferences into a single .json file. You can import this file later or on another device to restore your entire setup instantly.
