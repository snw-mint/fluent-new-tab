Theme & Appearance

Located at the very top of the settings menu, you can quickly toggle between three main color schemes for your new tab interface.

Light Mode

A clean, bright interface using the standard Fluent Design light palette. Ideal for daytime usage and well-lit environments.

Dark Mode

A deep, high-contrast dark palette that reduces eye strain in low-light environments and makes colors pop.

Auto Mode Default

Automatically syncs the extension's theme with your operating system's current color scheme (Windows/macOS settings).

Wallpaper

Customize the background of your new tab. If toggled off, the background will revert to a solid color based on your active theme.

Built-in Presets

Choose from three default, high-quality abstract wallpapers carefully designed to match the Fluent Design aesthetic without distracting from your content.

Upload Custom Image

Click the upload icon to set any personal image as your background. Images are automatically optimized, converted to WebP format, and stored securely in your browser's local database.

Dynamic Image Sources

Instead of a static image, you can fetch a new wallpaper automatically every day from external sources: Bing Daily Image, NASA Astronomy Picture of the Day (APOD), or Wikimedia Commons. Credits are shown in the bottom left corner.

Greeting

The large, centralized text that welcomes you every time you open a new tab. It dynamically changes based on the time of day (Morning, Afternoon, Evening, Night).

Display Name

Enter your name to receive personalized greetings (e.g., "Good morning, Alex!"). Leave it blank for a standard, universal greeting.

Icon Style

Choose the aesthetic of the time-of-day icon displayed next to the greeting text. You can use the colorful 3D Fluent emojis, or a sleek, monochromatic Outline style.

Weather

A subtle widget displayed in the top-right corner that shows the current temperature and weather conditions for your location, powered by Open-Meteo.

Location Setup

Type the name of your city in the input field and click the save icon. The system will geocode the location and fetch the current meteorological data.

Temperature Unit

Toggle seamlessly between Celsius (°C) and Fahrenheit (°F). The conversion is applied instantly without needing to fetch data again.

Shortcuts

The grid of quick links located right below the search bar. You can add, edit, and drag-and-drop to reorder them directly on the main page.

Grid Rows

Control the maximum amount of shortcuts displayed on the screen. You can limit the grid to show anywhere from 1 to 4 rows. If disabled entirely, the grid is hidden to keep the interface completely minimalist.

App Launcher

A floating button on the left side of the screen that opens a quick-access drawer to web applications.

Ecosystem Provider

Select which suite of tools you use the most. The launcher will automatically populate with shortcuts and icons for Microsoft 365 (Word, Teams, Copilot), Google Workspace (Docs, Drive, Gemini), or Proton AG (Mail, Calendar, Pass).

Search

Customize the behavior and appearance of the central search bar. You can change the default search engine directly on the main page by clicking the engine icon inside the bar.

Suggestions

When enabled, typing in the search bar will fetch live autocomplete suggestions to help you find what you are looking for faster.

Clear Search (Google)

This setting appends the &udm=14 parameter to Google searches. It forces Google to return a clean, classic list of links, stripping away AI overviews, sponsored widgets, and shopping cards.

Compact Bar

Reduces the height and visual footprint of the search bar for a tighter, more refined aesthetic.

Voice Search

Enables the microphone icon inside the search bar, allowing you to search the web using voice recognition (requires browser permission).

Advanced & Backup

Reduce Motion

Disables all CSS transitions, interface animations, and hover effects. This is highly recommended for older computers to improve performance, or for users who prefer an instant, snappy response.

Language

Force the interface language of the extension menu and greetings. It supports over 20 languages. By default, it follows your browser's language settings.

Data Backup (Export / Import)

Allows you to export all your configured settings, shortcuts, and preferences into a single .json file. You can import this file later or on another device to restore your entire setup instantly.
